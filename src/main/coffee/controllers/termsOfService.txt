利用規約

このGAS Library Boxサービス(以下GLB)、GAS Library Box Chrome Extension(以下GLB Extension)を利用していただく場合、以下の利用規約に同意していただく必要があります。

0. 定義

GLB : 本サービス
GLB サーバ : GLBにて利用する情報を保存するためのサーバ
GLB ユーザ : GLBの利用者
GLB アカウント : GLB上にライブラリを公開するために必要なGLB上に作成されたアカウント

1. GLBとは

GLBはApps Scriptのライブラリ（以下ライブラリ）を管理したり、ライブラリを公開、共有、検索するためのサービスです。

2. GLBの利用について

GLBを利用する上で発生する如何なる事象もGLBは責任を負わず、保証をしません。
GLBの利用は利用者の自己責任で実施してください。

2. GLBアカウントとは

GLBを利用してライブラリを公開するには、GLBにアカウントを登録する必要があります。
ライブラリの管理や、GLBにて公開されているライブラリの検索だけを行う場合はアカウント登録は必要ありません。

ユーザ登録にはGoogleアカウントのメールアドレス、Google のプロフィール URLが必要になりますが、
GLBはそこで得たメールアドレスはユーザ認証の為だけ、それ以外の目的で利用しません。
Google のプロフィールURLについては、ライブラリの公開を行った場合、
ライブラリの詳細とともに、すべてのユーザに公開されます。

3. Managing Library

GLBを利用して、ライブラリの管理を行う場合、ライブラリはGLB サーバには保存されず、GLB Extensionを利用しているChromeに保存されます。

4. Publishing Library

GLBを介してライブラリを公開する場合、そのライブラリの以下の情報はGLB上ですべてのGLB利用者に公開されます。

  * 作成者名（アカウント登録時に設定するニックネーム）
  * 作成者画像（Google 上にて公開されているユーザのアイコン ただし非公開も可能）
  * 作成者のGoogle  プロフィールURL
  * 登録日、登録情報の修正日
  * ライブラリのプロジェクトキー
  * ライブラリの名称（ライブラリ公開時に設定）
  * ライブラリの詳細（ライブラリ公開時に設定）

なおGLBはGLBの使用者がGLBに保存したライブラリその物の所有権を主張しません。
ただしGLBがGLBユーザに明らかに危害が関わるライブラリと判断した場合は、
GLBサーバ上から情報を削除する権利を持ちます。

5. Using Published Library

GLBを介して公開されているライブラリの利用は利用者の自己責任です。
GLBはライブラリを利用して発生した如何なる事象も責任を負わず、保証もいたしません。

6. Changing terms of Service

本規約を随時変更することができるものとします。
変更の内容についてはGLBサーバ上に２週間表示した時点で、すべてのユーザが承諾したものとみなします。
